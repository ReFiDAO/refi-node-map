# ReFi DAO Journey Microsite

A premium, interactive microsite showcasing the ReFi DAO journey from 2024 to 2025.

## Features

- **Vertical Timeline**: Elegant vertical layout with staggered scroll animations.
- **Premium Design**: Dark mode, noise textures, glassmorphism, and smooth gradients.
- **Interactive Elements**: Scroll-triggered reveals, hover effects, and expandable content.
- **Responsive**: Fully responsive design for mobile and desktop.

## Tech Stack

- **Core**: HTML5, CSS3, Vanilla JavaScript
- **Styling**: Custom CSS with CSS Variables
- **Icons**: Lucide Icons (CDN)

## Getting Started

No build process required.

1. Clone the repository.
2. Open `index.html` in your browser.
3. Or view the live site at: [https://johnellison.github.io/refi-dao-microsite/](https://johnellison.github.io/refi-dao-microsite/)

## Project Structure

- `index.html`: Main structure and content
- `style.css`: All styles and animations
- `script.js`: Scroll logic and interactivity
- `public/`: Images and static assets
